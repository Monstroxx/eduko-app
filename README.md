# Eduko — App

Modern, open-source school management app. Built with Flutter for mobile & web.

## Features

- Timetable view with lesson content, rooms, teachers
- Absence & excuse management
- Attendance overview
- Substitutions & cancellations (color-coded)
- Appointments & exams
- Offline-first with background sync
- i18n support (German first, extensible)

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Framework | Flutter |
| State | Riverpod |
| Navigation | go_router |
| Local DB | Drift (SQLite) |
| HTTP | dio |
| i18n | flutter_localizations + intl |

## Platforms

- Android
- iOS
- Web

## Getting Started

```bash
flutter pub get
flutter run
```
